# Projeto Formulário de Cadastro em React

Este projeto demonstra um formulário de cadastro simples utilizando React.
## Funcionalidades

- Cadastro com campos de Nome, Email e Senha
- Validação de campos obrigatórios
- Exibição de mensagem de erro caso algum campo não seja preenchido
- Limpeza dos campos após cadastro realizado

## Como executar

1. Instale as dependências:
   ```
   npm install
   ```
2. Inicie o projeto:
   ```
   npm start
   ```
3. Acesse `http://localhost:3000` no navegador.

## Estrutura do projeto

- `src/components/CadastroForm.jsx`: Componente principal do formulário
- `src/App.jsx`: Importa e exibe o formulário
